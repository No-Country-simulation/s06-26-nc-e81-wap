# S06-26-NC-E81-WAP

# App de Orientación Personal con IA — Orienta360 (B2C)

## 📌 Descripción

Ecosistema web de orientación personal impulsado por IA, diseñado para apoyar a personas de grupos sub-representados en cinco dimensiones esenciales: **formación**, **empleabilidad**, **experiencias**, **mentorías** y **salud mental**.

No es solo una app de empleos. No es solo una plataforma de cursos. Tampoco es solo una solución de bienestar. Es un ecosistema que mira a cada persona de forma 360°, con empatía, acompañamiento y relevancia cultural — reuniendo en un solo lugar el soporte necesario para que cada persona pueda desarrollarse, pertenecer y avanzar con más oportunidad y confianza.

## 🎯 Funcionalidades clave

- 🧭 **Orientar (agente IA)** — Analiza el perfil profesional contra el mercado, muestra el gap porcentual y recomienda una trayectoria concreta para cerrarlo.
- ❤️ **Salud (check-in emocional)** — Check-in diario con emoji. El agente IA interpreta el estado emocional, sugiere acciones humanas y deriva al CVV en situaciones de crisis.
- 📚 **Formaciones** — Cursos gratuitos y trayectorias personalizadas basadas en el gap del perfil.
- 💼 **Empleabilidad** — Match automático entre perfil y vacantes con gap visible: "Cumples el 70% — ve qué falta y cómo resolverlo".
- 🎤 **Experiencias** — Eventos en vivo y grabados con testimonios de personas que superaron barreras similares.
- 🤝 **Mentorías** — Networking humanizado con mentores que invitan a una práctica, no solo a una entrevista formal.

## 🧩 Alcance del proyecto

- Frontend: **Vite + React + TypeScript + Tailwind CSS** (shadcn/ui, react-router-dom v7, TanStack Query, Zustand, i18n)
- Agente IA integrado en el frontend con mocks
- Backend / API REST (en desarrollo)
- Soporte multilingüe: ES, PT, EN
- Diseño responsivo (PWA-ready)

## 🎯 Objetivo del sistema

El sistema busca ofrecer un ecosistema integrado de orientación personal que actúe simultáneamente en cinco dimensiones clave:

- **Formación** — Cursos y trayectorias personalizadas según el perfil y gap detectado.
- **Empleabilidad** — Match con vacantes y visualización clara del gap a cubrir.
- **Experiencias** — Testimonios y eventos con referentes reales que inspiraron.
- **Mentorías** — Conexión humanizada con mentores del mercado.
- **Salud mental** — Check-in diario con IA y derivación a soporte profesional en crisis.
